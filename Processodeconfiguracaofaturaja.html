1.Configuração Backend

2.Clonar o repositorio

Git clone endereco_do_repositorio


3.Instalar dependências

Composer install


4.Configuração do ambiente (env)

cp .env.example .env


5.Chave da aplicação

php artisan key:generate


6.Configuração da base de dados (env)

Definir as variáveis de ambiente no arquivo .env, conforme abaixo:

    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=example
    DB_USERNAME=root
    DB_PASSWORD=opcional


7.Criação das tabelas

php artisan migrate


8.População da base de dados (se houver dados iniciais)

php artisan db:seed


9.Criação do link simbólico para armazenamento

php artisan storage:link


10.Limpeza de cache

php artisan config:clear


11.Iniciar o servidor

php artisan serve

php artisan serve --host=0.0.0.0 --port=8000


12.Configuração adicional

Definir as variáveis de ambiente no arquivo .env, conforme abaixo:

    TENANT_DB_HOST=127.0.0.1
    TENANT_DB_PORT=3306
    TENANT_DB_USERNAME=root
    TENANT_DB_PASSWORD=

13.Configuração da URL da aplicação(APP_URL)

APP_URL=http://localhost:port


14. Lista os IPs mais comuns — o cors.php aceita todos os IPs locais automaticamente

SANCTUM_STATEFUL_DOMAINS=localhost:3000,localhost:port,127.0.0.1:3000,127.0.0.1:8000,192.168.1.120:3000,192.168.1.120:8000


15.FRONTEND

    1.VITE_APP_NAME="${APP_NAME}"
    2.VITE_APP_URL="${APP_URL}"(IP&PORT)
    3.VITE_FRONTEND_URL="http://localhost:3000"


16. AGT - Assinatura RSA

Configuração Frontend

1.Instalar dependências

npm install


2.Criar arquirvo .env.local


3.Definição das variáveis

APP

    NEXT_PUBLIC_APP_NAME=
    NEXT_PUBLIC_APP_ENV=local

BACKEND (LARAVEL)

    NEXT_PUBLIC_API_URL=http://localhost:port
    NEXT_PUBLIC_BACKEND_URL=http://localhost:port

AUTH / SANCTUM

    NEXT_PUBLIC_CSRF_URL=http://localhost:port/sanctum/csrf-cookie
    NEXT_PUBLIC_AUTH_COOKIE=true

4.Iniciar o servidor

npm run dev
